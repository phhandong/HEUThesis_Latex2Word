# 生成 Word 与 GraduateEssay 格式差异

审计日期：2026-07-11

## 1. 比较对象

- 标准稿：`GraduateEssay.docx`
- 生成稿：当前转换脚本从
  `HeuThesis_Overleaf/examples/book/bachelor/main.tex` 生成，参数为
  `--degree master`
- 检查方式：Word 16 有效格式、OOXML、全文 PDF 渲染、代表页视觉检查

重要限制：`GraduateEssay.docx` 是本科论文；当前 CLI 仅支持 `master`、
`doctor`。封面页序、学位称谓、新版声明、成果标题、奇数页开章及空白背面，
不能直接判定为转换缺陷。

## 3. 差异总表

严重度：P0 = 阻止交付；P1 = 明显格式错误；P2 = 兼容性或一致性问题；
P3 = 低风险清理。

| 严重度 | 区域 | GraduateEssay | 当前生成稿 | 影响 | 建议任务 |
| --- | --- | --- | --- | --- | --- |
| P0（已解决） | 目录保存结果 | 已缓存 53 条目录结果 | 已由 Word 两遍更新并缓存真实目录；使用原生 `TOC1–3` 样式 | 普通打开即可看到目录；最终样例 72 页 | 保留目录缓存和样式 ID 回归测试 |
| P0（已解决） | 公式回退 | 公式完整显示，带按章编号 | Pandoc 数学兼容预处理后无转换警告；公式页无源码回退和缺字方框 | 阻断影响已消除 | 保留不支持命令及代码区保护测试 |

## 4. 学位类型或模板策略差异

以下项目先不列为缺陷：

| 项目 | 原因 |
| --- | --- |
| 本科/工程硕士页眉称谓 | 学位类型不同 |
| 分类号、UDC、编号、校外导师、英文扉页 | 硕士封面结构 |
| 新版 AI 辅助内容声明 | 声明模板版本不同 |
| 攻读学士/硕士期间成果标题 | 学位类型不同 |
| 章节奇数页开章及空白背面 | 双面打印策略 |
| 9 节与 22 节 | 内容和硕士分节策略不同 |

注意：标准稿自身第 45 页存在“页眉第五章、正文第四章”的错误。生成稿动态
`STYLEREF` 页眉反而正确。不可机械复制标准稿全部内部结构。
